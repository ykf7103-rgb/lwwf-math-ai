# 小數乘法與簡易方程學習指南

本學習指南旨在協助學習者掌握小數乘法的運算技巧、估算方法，以及如何解與乘除法相關的一步簡易方程。內容涵蓋核心概念說明、實戰練習與深度探究。

---

## 第一部分：核心概念解析

### 1. 小數乘法（一位小數與兩位小數）

小數乘法的關鍵在於確定「積」（相乘的結果）的小數位數。

*   **積的小數位數規律：**
    *   **一位小數 × 一位小數：** 其積為「兩位小數」。
        *   *原理：* $\frac{4}{10} \times \frac{2}{10} = \frac{8}{100}$。分母為 100，故積為兩位小數（如：$0.4 \times 0.2 = 0.08$）。
    *   **兩位小數 × 一位小數：** 其積為「三位小數」。
        *   *原理：* $\frac{235}{100} \times \frac{7}{10} = \frac{1645}{1000}$。分母為 1000，故積為三位小數（如：$2.35 \times 0.7 = 1.645$）。
*   **計算要點：**
    *   使用直式計算時，先當作整數計算，最後再補上小數點。
    *   若積的位數不足，必須在前補上「0」和小數點。
*   **估算技巧：**
    *   在精確計算前，可先將小數化為最接近的整數進行估算。
    *   *範例：* $9.7 \times 5.2 \approx 10 \times 5 = 50$。實際積應接近 50。

### 2. 簡易方程（二）：乘法與除法的一步方程

解方程的目標是利用「等式性質」使等號左邊只剩下未知數。

*   **等式性質（乘除法）：**
    *   在方程等號左右兩邊同時除以相同的數（0 除外），左右兩邊的值保持相等。
    *   在方程等號左右兩邊同時乘以相同的數，左右兩邊的值亦保持相等。
*   **解題與驗算步驟：**
    1.  **依題意建立方程：** 根據問題設定未知數（如：設原價為 $a$ 元）。
    2.  **解方程：** 運用等式性質求解。
    3.  **驗算：** 將求得的解代入原方程的「左邊」或「右邊」，確認其值是否等於另一邊。
    4.  **寫出答案：** 根據題目要求寫出完整的答案。

## 第二部分：核心詞彙表

| 詞彙 | 定義與說明 |
| :--- | :--- |
| **小數 (Decimal)** | 包含整數部分、小數點和小數部分的數值。 |
| **積 (Product)** | 兩個或多個數相乘得到的結果。 |
| **估算 (Estimation)** | 在精確計算前，透過化簡數值來預測結果大約是多少的過程。 |
| **方程 (Equation)** | 含有未知數的等式，如 $3r = 60$。 |
| **未知數 (Unknown)** | 方程中代表數值的英文字母（如 $x, y, r, u$ 等）。 |
| **等式性質** | 在方程兩邊同時進行相同的加、減、乘、除運算，等式依然成立。 |
| **驗算 (Verification)** | 將求得的數值代入原問題中核對，以確保解的正確性。 |

## 參考答案要點（摘錄自教材內容）

*   **小數乘法：** $0.4 \times 0.2 = 0.08$；$2.35 \times 0.7 = 1.645$。
*   **方程解：** $u = 7$；$w = 6$；$f = 60$；$v = 140$。
*   **應用題：**
    *   鉛筆刨：$3r = 60, r = 20$ 克。
    *   測驗分：$15y = 75, y = 5$ 分。
    *   吸塵機：$a - 150 = 399, a = 549$ 元。
    *   走廊長度：因為「來回一次」等於走廊長度的 2 倍，來回 3 次即為 6 倍長度，故 $6y = 48, y = 8$ 米。
